# 🎬 Netflix Movie Data Analysis (EDA)

## 📌 Overview
This project performs Exploratory Data Analysis (EDA) on a movie dataset to uncover patterns in genres, popularity, ratings, and release trends.

The analysis demonstrates core data analyst skills including data cleaning, transformation, visualization, and insight generation.

---

## 🎯 Objectives
- Identify the most frequent movie genres
- Analyze vote averages and popularity
- Find the most and least popular movies
- Explore movie release trends over time

---

## 🛠 Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 🧹 Data Preparation
- Removed irrelevant columns
- Handled missing values and duplicates
- Converted release dates to year format
- Processed multi-genre entries
- Categorized ratings using quartiles

---

## 📊 Key Insights
- Drama is the most common genre
- Popularity is highly skewed toward a few movies
- Ratings cluster in mid-to-high ranges
- Movie releases increased in recent years

---

## ▶️ How to Run

pip install pandas numpy matplotlib seaborn
